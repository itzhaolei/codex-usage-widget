## Quota Bubble v2.2.0

This release is available for macOS. The latest Windows package remains v2.1.3.

### Highlights

- Replaces the second quota row with a reset-credit expiration list that grows with the window.
- Shows red status dots for reset credits expiring within three days and green dots otherwise.
- Adds the current Codex account and subscription expiration below the metric cards on macOS.
- Keeps account and subscription display values in memory instead of adding them to the quota snapshot.
- Uses compact countdown formatting and omits zero-value time units.
- Fixes transient empty-state flicker and Pro20x plan detection.
- Stabilizes quota percentages and reset countdowns across live API and session-log fallback sources.

### Install

Download `QuotaBubble-2.2.0-macOS-Installer.zip`, unzip it, and open `Install Quota Bubble.app`.
