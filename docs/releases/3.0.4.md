## Quota Bubble v3.0.4

This release removes the macOS runtime dependency on Node.js and improves subscription labels and in-app updates.

### Native macOS quota service

- Fetches quota, balance, plan, and reset-credit data directly inside the SwiftUI application.
- Requires no Node.js, npm, separately installed Codex CLI, Xcode, or terminal tools at runtime.
- Preserves account-isolated snapshots, same-cycle quota stabilization, reset-credit caching, and token-free persisted data.
- Ships as a Universal Binary for Apple Silicon and Intel Macs, targeting macOS 13 or later.

### Updates and plan labels

- Shows download progress and percentage while installing a macOS update.
- Automatically restarts Quota Bubble after a successful macOS update.
- Displays the generic Codex `pro` plan as `Pro20x`, while preserving explicit `Pro5x` plans on macOS and Windows.

### Install

- macOS: download `QuotaBubble-3.0.4-macOS-Installer.zip`, unzip it, and open `Install Quota Bubble.app`.
- Windows: download `QuotaBubble-3.0.4-Windows-Setup.exe` and run the graphical installer.
