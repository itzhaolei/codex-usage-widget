## Quota Bubble v3.0.1

This macOS-only maintenance release improves updater reliability, native window behavior, installation compatibility, and the SwiftUI glass interface. Windows remains on v2.1.3.

### Highlights

- Makes update checks resilient to GitHub API rate limits with a public Release-page fallback.
- Adds download retries, HTTP error detection, `ditto` extraction, installer diagnostics, and a no-Python Dock fallback for clean Macs.
- Installs Quota Bubble in `/Applications` and restores complete Finder and Dock icon registration.
- Uses native Liquid Glass on macOS 26 with an adaptive translucent fallback on macOS 13 through 15.
- Keeps Universal release builds compatible with older Xcode SDKs while preserving native Liquid Glass when built with Xcode 26.
- Keeps pinned windows above other windows only in their current macOS Space, without following users into other desktops or full-screen apps.
- Keeps language menus open during one-second quota refreshes and restores reliable pin/unpin state binding.
- Prevents restarts from leaving an application process without a visible window.
- Improves progress-bar dimensions and keeps localized metric cards equal-height, top-aligned, and dynamically sized.
- Removes the obsolete Codex CLI prerequisite overlay so the widget opens directly.

### Install

Download `QuotaBubble-3.0.1-macOS-Installer.zip`, unzip it, and open `Install Quota Bubble.app`.
