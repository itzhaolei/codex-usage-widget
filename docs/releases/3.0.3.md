## Quota Bubble v3.0.3

This release replaces the Windows PowerShell prototype with a real, self-contained desktop application and graphical installer.

### Windows desktop application

- Compiled with .NET 8 and WPF as a native Windows GUI application.
- Runs as one application process with a synchronized taskbar icon, window, and tray menu.
- Preserves the macOS-aligned quota layout, plan badge, progress bar, reset-expiration rows, balance, account, subscription expiration, themes, pinning, saved position, and ten languages.
- Reads the current Codex account and usage endpoint directly inside the application, with one-second refresh and account-isolated in-memory fallback.
- Requires no PowerShell interaction, Node.js, npm, terminal commands, or separately installed .NET runtime.

### Installer and updates

- Download `QuotaBubble-3.0.3-Windows-Setup.exe` and double-click it.
- The graphical installer creates Start menu shortcuts, optionally creates a desktop shortcut, and can launch Quota Bubble at sign-in.
- Updates download and launch the latest graphical installer from inside Quota Bubble.
- Windows Settings and the Start menu provide normal graphical uninstall entry points.

### Account safety

- Revalidates the active account after every quota response, preventing an in-flight response from the previous account from appearing after a switch.
- Keeps fallback quota only for the same irreversible account fingerprint.
- Does not persist Codex access tokens, raw account identifiers, or personal account data.

### macOS

The macOS application is repackaged as v3.0.3 so both platforms remain on the same release line. Its behavior is unchanged from v3.0.2.
