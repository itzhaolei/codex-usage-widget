## Quota Bubble v3.0.8

This release makes weekly quota resets stable and adds a polished recharge animation driven only by real Codex quota data.

### Added

- Animates a genuine weekly quota recharge from the previous displayed percentage to the new value.
- Coordinates the progress sweep, color-card particles, orbit tracks, and selected progress color without replacing the normal progress-bar style.
- Provides five persistent progress colors with subtle moving star highlights.

### Fixed

- Prevents stale local snapshots and fresh API data from alternating during the short synchronization window after a quota reset.
- Requires the same account fingerprint and a newer weekly reset cycle before playing the recharge animation.
- Handles Codex responses that expose the weekly quota as the only usage window.
- Removes all preview-only recharge triggers and simulated quota values from production builds.

### Install

- macOS: download `QuotaBubble-3.0.8-macOS-Installer.zip`, unzip it, and open `Install Quota Bubble.app`.
- Existing macOS users can also choose **Check for Updates** from the application menu.
