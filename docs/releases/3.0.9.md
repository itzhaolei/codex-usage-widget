## Quota Bubble v3.0.9

This macOS release adds synchronized multi-window support while keeping Quota Bubble in one native application process.

### Added

- Adds a localized **New Window** command to the application menu.
- Supports `Command-N` for creating additional quota windows.
- Shares one live quota store and one refresh timer across every window, keeping quota, account, balance, reset credits, and subscription data synchronized.

### Changed

- Keeps theme, pin state, progress color, and position independent for each window.
- Binds per-window preferences to persistent scene identifiers so macOS restoration order cannot exchange settings between windows.

### Fixed

- Makes the custom close control immediately hide and unregister only the selected borderless window.
- Exits the application and clears its Dock running state after the final window closes.

### Install

- macOS: download `QuotaBubble-3.0.9-macOS-Installer.zip`, unzip it, and open `Install Quota Bubble.app`.
- Existing macOS users can also choose **Check for Updates** from the application menu.
