## Quota Bubble v3.0.6

This release improves the native macOS glass treatment, adds sharing flows, and simplifies graphical installer downloads from the website.

### Added

- Added localized Official Website and Share items to the macOS application menu.
- Added a localized website sharing section with system sharing and one-click link copying.
- Added operating-system detection so the website downloads the matching macOS or Windows graphical installer directly.

### Changed

- Switched the macOS 26 light theme and metric cards to native clear Liquid Glass, with a low-opacity fallback on older systems.
- Removed the active background tint from the pin button so its green icon alone indicates the pinned state.
- Redesigned the website around the current SwiftUI interface and removed command-line installation guidance.

### Install

- macOS: download `QuotaBubble-3.0.6-macOS-Installer.zip`, unzip it, and open `Install Quota Bubble.app`.
- Windows: download `QuotaBubble-3.0.6-Windows-Setup.exe` and run the graphical installer.
