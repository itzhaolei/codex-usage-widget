## Quota Bubble v3.0.0

This major release rebuilds the macOS client as one native SwiftUI app. The floating window, Dock icon, menus, updater, and lifecycle now belong to the same process. Windows remains on v2.1.3.

### Highlights

- Replaces the legacy AppKit HUD and separate Dock launcher with one SwiftUI `Quota Bubble.app`.
- Preserves the established dark glass HUD, fixed width, dynamic reset-expiration height, theme, pin state, and saved position.
- Left-aligns the quota row, progress bar, reset-expiration list, metric cards, and account details.
- Keeps one-second local quota refreshes while retaining the last valid snapshot during transient writes.
- Validates a redacted account fingerprint before displaying quota data after an account switch.
- Preserves update progress while replacing the installed app bundle and reports the final result.
- Provides a Universal macOS binary for Apple Silicon and Intel Macs, targeting macOS 13 or later.
- Installs the prebuilt release app from the one-line installer without requiring Xcode Command Line Tools.
- Adds repeatable model, account-switch, shell, strict-concurrency, and SwiftUI build checks.

### Install

Download `QuotaBubble-3.0.0-macOS-Installer.zip`, unzip it, and open `Install Quota Bubble.app`.
