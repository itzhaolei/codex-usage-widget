## Quota Bubble v3.0.7

This maintenance release fixes the macOS distribution build so the native Liquid Glass implementation is preserved in the downloaded application.

### Fixed

- Builds the macOS universal installer with Xcode 26.6 on the GitHub macOS 26 runner.
- Verifies that the packaged application contains native SwiftUI Glass symbols before a release can be uploaded.
- Restores clear Liquid Glass in light mode on macOS 26; older macOS versions continue to use the compatible blur material.

### Install

- macOS: download `QuotaBubble-3.0.7-macOS-Installer.zip`, unzip it, and open `Install Quota Bubble.app`.
- Windows: download `QuotaBubble-3.0.7-Windows-Setup.exe` and run the graphical installer.
