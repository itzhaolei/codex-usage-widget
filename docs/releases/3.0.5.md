## Quota Bubble v3.0.5

This maintenance release fixes the macOS fallback update download URL.

### Fixed

- Converts parsed version components back into dotted text before building the fallback installer URL.
- Prevents GitHub API fallback updates from producing an array-style filename such as `[3, 0, 4]` and the resulting `bad range in URL` error.
- Validates fallback release tags before constructing a download URL.

### Install

- macOS: download `QuotaBubble-3.0.5-macOS-Installer.zip`, unzip it, and open `Install Quota Bubble.app`.
- Windows: download `QuotaBubble-3.0.5-Windows-Setup.exe` and run the graphical installer.
