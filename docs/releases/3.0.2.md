## Quota Bubble v3.0.2

This cross-platform maintenance release fixes cross-account quota contamination and brings the Windows client in line with the current macOS data model and interface.

### Fixed

- Binds authenticated quota snapshots to a local irreversible account fingerprint without storing the raw account identifier.
- Prevents account-less session-log rate limits from being assigned to the currently authenticated account.
- Keeps the current account's last valid live quota during a temporary endpoint failure instead of replacing it with a previous account's session value.
- Clears old quota immediately after an account switch and waits for data verified for the new account.
- Adds regression tests for both endpoint-failure stability and account-switch isolation.

### Windows

- Adds the current weekly quota row, reset-credit expiration list, balance, available reset count, account, subscription expiration, and plan badge.
- Adds ten languages with system-language detection and persistent tray language selection.
- Adds persistent theme, pin, position and language state, dynamic reset-row height, single-instance control, and in-app updates.
- Removes the obsolete Codex CLI setup overlay and uses the same account-isolated snapshot logic as macOS.

### Install

Download `QuotaBubble-3.0.2-macOS-Installer.zip`, unzip it, and open `Install Quota Bubble.app`.

For Windows, download `QuotaBubble-3.0.2-Windows.zip`, unzip it, then run `windows/install.ps1` with PowerShell.
