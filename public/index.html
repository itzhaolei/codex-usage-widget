Local macOS quota monitor for Codex

Your Codex quota, always visible.

Quota Bubble is a small floating macOS window that keeps Codex usage, weekly limits, balance, reset credits, and update status in sight without opening settings.

Always on topPin the bubble where you work and keep Dock state synced with the window.
Live quotaRefreshes local Codex quota, reset time, balance, and reset credits every second.
Private by designRuns locally and stores only redacted account fingerprints, never raw account IDs or tokens.
Easy updatesUse the app menu to check GitHub releases and update directly from the installer package.
